# ⚡ Code Compiler & Practice Platform

> A modern browser-based coding platform to practice programming, run code instantly, and improve typing + problem-solving speed — all in one place.

---

## 🚀 Features

- 🧠 Practice Mode – Solve curated coding problems (Easy → Medium)
- ⚡ Instant Code Execution – Run code in multiple languages
- 📊 Performance Tracking – Monitor runs, speed, and points
- ⌨️ Typing Practice Integration – Improve WPM alongside coding
- 🎯 Exercise-Based Learning – Structured problem list
- 🌙 Modern Dark UI – Clean & distraction-free interface
- 🔄 Multi-language Support:
  - Python 🐍
  - JavaScript ⚡
  - C++ 💻
  - Java ☕
  - Go 🚀
  - Rust 🦀
  - and more...

## ⚙️ Setup & Installation

```bash
# Clone the repo
git clone https://github.com/Devvvkun/Code_Compiler.git

# Navigate into folder
cd your-repo-name

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## 🔌 Environment Variables

Create a `.env` file in root:

```env
VITE_PISTON_API_URL=https://emkc.org/api/v2/piston
```

## 📌 Usage

1. Select a programming language  
2. Choose an exercise  
3. Write your code  
4. Click **Run ▶️**  
5. Submit & track performance
